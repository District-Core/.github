# About

District Core is an independent game development organization focused on creating cinematic, story-driven experiences powered by Unreal Engine 5.

Our goal is to deliver worlds that feel alive through cutting-edge technology, memorable characters, atmospheric environments, and meaningful gameplay.

---

# What We Build

- AAA-quality game prototypes
- Story-driven Action RPGs
- Unreal Engine tools & plugins
- Gameplay systems
- Visual effects
- Audio pipelines
- Internal development frameworks
- Web platforms for our projects

---

# Current Project

## 🌊 Tides of Aethra

A third-person Action RPG set in a world where elemental civilizations struggle against the expansion of the Metal Empire.

Features include:

- Deep narrative
- Dynamic combat
- Exploration
- Boss encounters
- Cinematic storytelling
- Unreal Engine 5 technologies
- Custom gameplay systems
